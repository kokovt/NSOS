# NSOS

My personal custom OS for the raspberry PI, in which is going to be partially based on windose20 in the future.

## 32 Bit systems:

32 Bit systems are planned to be *somewhat supported*, although its an afterthought. In the future I will comeback and rerwrite all of the 32 bit code in the future; but just know the *real OS is the 64 bit. The 32 bit is just there so I *can run it on a 0w if I really wanted to*.

## Plan for x86/64?

Maybe, in the future. I just wanted something I could do while gone for a few weeks on a raspberry pi 400.
